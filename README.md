# README

steps necessary to get the
application up and running.

*clone the folder on your computer
$ git clone 

*go to folder
$ cd Eventbrite

$ bundle install

*create BDD
 $ rails db:create 
 
*Active the migrations and run the seed :

$ rails db:migrate $ rails db:seed

$ rails server
*go to 
http://localhost:3000/
